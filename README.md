# Client Panel - Artist Dashboard

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm preview
```

## Features

- Artist authentication with OTP verification
- Dashboard with release stats and revenue overview
- Release management (create, edit, view releases)
- WAV audio file upload for tracks
- Revenue analytics with charts
- Profile and bank details management
- PWA support for offline access

## Pages

- Login - Artist login page
- Register - Artist registration with OTP
- VerifyOTP - OTP verification page
- Dashboard - Overview with stats and quick actions
- Releases - List of all releases with status
- CreateRelease - Create/edit releases with track upload
- Revenue - Revenue analytics and breakdown
- Profile - Artist profile and bank details management

## PWA Installation

The dashboard is configured as a PWA. Users can install it:
- On desktop: Click the install icon in the browser address bar
- On mobile: Use the "Add to Home Screen" option from the browser menu
